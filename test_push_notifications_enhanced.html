Configuration Requirements

Firebase Setup Required

You need to configure Firebase server key in your .env file:

# Add to your .env file:
FIREBASE_SERVER_KEY=your_actual_firebase_server_key_here
FIREBASE_PROJECT_ID=connect-app-fbaca
FIREBASE_DATABASE_URL=https://connect-app-fbaca.firebaseio.com

To get your Firebase Server Key:

  1. Go to Firebase Console → Your Project → Project Settings
  2. Click on "Cloud Messaging" tab
  3. Copy the "Server key" from the "Project credentials" section
  4. Update the FIREBASE_SERVER_KEY in your .env file
